Lockdown.sh

23-05-2020

Lockdown.sh is a simple POSIX script created to harden a new linux install. While learning more about linux and server admin, I came across a lynis. Which is an incrediblly neat tool to audit a linux server.

I started by creating a new debian 10 VPS and running

$ lynis audit system

This performs various checks on the current system and reports warnings and suggestions. The initial hardening index I got was

Hardening index : 57 [###########         ]

Then after working through the suggestions and applying fixes I managed to increase that to 88.

Hardening index : 88 [#################   ]

As I was doing so I wrote up the changes I was making in this script.

Now creating a new instance and running this script applies all the changes.
